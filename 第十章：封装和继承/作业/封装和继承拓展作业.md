# 封装和继承拓展作业

```Python
创建一个名为Animal的父类，这个类包含一个方法eat()，在调用时输出"Eating..."。
然后创建一个名为Dog的子类，这个类继承自Animal类，并添加一个方法bark()，在调用时输出"Barking...".
```

```Python
创建一个叫做Person的父类，包含name和age两个属性，
以及一个介绍自己（即打印自己名字和年龄）的方法introduce().
然后创建一个叫做Student的子类，继承自Person类，添加name，age，grade属性和学习（即打印正在学习）的方法study().
```

```Python
创建一个名为Employee的父类，具有name, salary等属性和一个work()方法。
然后创建两个子类Manager和Programmer，它们都继承自Employee类，
但是work()方法的行为应该不同
（例如，对于Manager，work()可能会输出"管理员工"，
对于Programmer，work()可能会输出"编写代码"）。
```
