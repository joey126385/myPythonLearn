## 分支结构（逻辑判断判断）

```python
关键词： 
if 如果   else否则    elif否则如果

理解：
如果今天是星期天，那么我们可以在家休息，否则我们需要去上班。
如果今天是晴天，那么我们可以出去打篮球，否则如果今天是阴天，我们可以出去踏青，否则我们就在家里看电影。
```

### 单分支：

#### 语法结构：

```python
if 条件表达式:
    缩进相同的代码块，这个缩进是用于告诉计算机，这段代码是属于if的
```

#### 单分支案例1：

```python
'如果今天是星期天，那么我们可以在家休息'

# 1、从控制台输入今天星期几，并用变量将这个星期天数保存。
date = input("请输入今天星期几：")
# 2、判断今天是否是星期天
# Python 中 = 是赋值符号, ==是等于符号 , 用于写条件表达式(比较运算符), 用于判断 == 两变的值是否相等。
if date=='星期天':
    # 在满足条件之后,程序会执行 : 后面换行之后缩进一致的代码块
    print("在家休息")
    print("吃饭追剧打游戏！")
```

#### 单分支案例2：

```python
'如果今天是星期天，那么我们可以在家休息(如果现在是星期天的下午,我们可以点外卖)'

# 1、从控制台输入今天星期几，并用变量将这个星期天数保存。
date = input("请输入今天星期几：")
# 2、判断今天是否是星期天
# Python 中 = 是赋值符号, ==是等于符号 , 用于写条件表达式(比较运算符), 用于判断 == 两变的值是否相等。
if date=='星期天':
    # 在满足条件之后,程序会执行 : 后面换行之后缩进一致的代码块
    print("在家休息")
    print("吃饭追剧打游戏！")
    # 3、从控制台接受现在是什么时候？
    x = input("请输入现在是上午？中午？下午？晚上？")
    # 4、判断是否满足 是 下午？
    if x=='下午':
        print("我们可以点外卖")
        print("外卖真好吃！")

# Python 中的缩进,缩进很重要,在Python中用于去区分代码等级,不要随意缩进,根据情况进行缩进
```

### 双分支

#### 语法：

```python
if 条件表达式:
    满足条件执行，缩进一致代码块
else:
    满足条件执行，缩进一致代码块
```

#### 双分支案例1：

```python
'如果今天是星期天，那么我们可以在家休息，否则我们需要去上班。'

# 1、从控制台输入今天星期几，并用变量将这个星期天数保存。
date = input("请输入今天星期几：")
# 2、判断今天是否是星期天
# Python 中 = 是赋值符号, ==是等于符号 , 用于写条件表达式(比较运算符), 用于判断 == 两变的值是否相等。
if date=='星期天':
    # 在满足条件之后,程序会执行 : 后面换行之后缩进一致的代码块
    print("在家休息")
    print("吃饭追剧打游戏！")
else:
    # 否则 后面要做的事情也是在 : 后面换行,且缩进一致的代码块
    print("今天去上班")
    print("我爱上班！我是快乐打工人！")
```

### 多分支

#### 语法：

```python
if 条件表达式:
    满足条件执行，缩进一致代码块
elif 条件表达式:
    满足条件执行，缩进一致代码块
...
else:
    满足条件执行，缩进一致代码块
```

#### 多分支案例：

```python
"""
如果 【今天是晴天，那么我们可以出去打篮球，】
否则如果 【今天是阴天，我们可以出去踏青，】
否则如果 【今天是雪天，我们可以出去打雪仗，】
否则 【我们就在家里看电影。】
"""
# 否则如果   else否则 + if如果 = elif

# 1、从控制台去接收用户输入的天气，让这个程序的判断变的灵活
# 2、判断今天是否是晴天？
# 3、不满足 上面的条件 , 就继续判断 今天是否是阴天？
# 4、以上条件都不满足, 进入否则 执行否则的代码

tianqi = input("请输入今天的天气")

if tianqi=='晴天':    # 唯一的真爱
    print("可以出去打篮球")
elif tianqi=="阴天":    # 妃子1
    print("我们可以出去踏青")
```

## 循环结构

### while循环

#### 什么是while循环？

while 循环和 if 条件分支语句类似，即在条件（表达式）为真的情况下，会执行相应的代码块。不同之处在于，只要条件为真，while 就会一直重复执行那段代码块。

#### 语法结构?

```python
while 条件表达式:
    代码块
```

#### 条件表达式？

```python
条件表达式通常，结合一下三种运算符去进行编写。会有一个返回值，要么返回条件成立 True ，要么返回条件不成立 False

<、>、<=、>=、==（等于）、!=（不等于）   比较运算符    
条件表达式举例：
            5%2==1    返回True
            10/2==5    返回True
            2**3>=8    返回True
            5%2!=1     返回False

in （在...里面）、not in（不在...里面）    成员运算符
条件表达式举例：
        1 in [1,2,3,4,5]        返回True
        1 not in [1,2,3,4,5]    返回False

not（取反） 、 and（并且） 、 or（或者） 逻辑运算符
条件表达式举例：
        not 1+1==2     返回False
        1+1==2 and 1-1==0      返回True  
        1+1==3 and 1-1==0      返回False
        1+1==3 or 1-1==0      返回True    
```

#### while循环案例1：

```python
# 实现了让这个 根据天气判断今天干啥的 程序一直工作
while True:      #死循环
    tianqi = input("请输入今天的天气")
    if tianqi == '晴天':  # 唯一的真爱
        print("可以出去打篮球")
    elif tianqi == "阴天":  # 妃子1
        print("我们可以出去踏青")
    elif tianqi == "雪天":  # 妃子2
        print("我们可以出去打雪仗")
    else:  # 皇后（由于皇帝不喜欢皇后，这是最后的选择）
        print("我们就在家里看电影")
```

### for循环

**for 循环，它常用于遍历字符串、列表、元组、字典、集合等序列类型，逐个获取序列中的各个元素。**

```python
循环原理，是通过迭代数据的方式进行循环
有个盒子，里面有6个小球【0,1,2,3,4,5】 每次从这个盒子里面拿一个小球，直到把小球拿完       这种叫做迭代。
```

#### 语法?

```python
语法：
for 变量  in  可迭代数据:
    缩进一致的循环代码块

可迭代数据: 列表、元组、字符串、集合、字典、range(初始值,终点值,步长)
range(初始值,终点值,步长) 函数用于生成一个等差数列对象，这个等差数列从初始值开始，到终点值结束(不包括终点值)，间隔为步长。
```

#### 案例：

```python
#  让变量 i 重复从  in 后面的可迭代数据中拿值，每次拿一个，直到拿完
for i in {0,1,2,3,4,5}:
    print("你好 世界！",i)

# 在控制台打印 1-100的偶数
for i in range(2,101,2):
    print(i)

# 在控制台打印 1-100的奇数
for i in range(1,100,2):
    print(i)

# 在控制台打印 1-100的 5 的倍数
for i in range(5,101,5):
    print(i)

# 在控制台打印 100-1的偶数
for i in range(100,1,-2):    #  等差数列从 100 开始 ，到1结束（不包括），每次循环 -2
    print(i)

# 在控制台打印 100-1的奇数
for i in range(99,0,-2):
    print(i)
```
