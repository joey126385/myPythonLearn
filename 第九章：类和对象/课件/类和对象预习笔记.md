## 类和对象

```python
# 什么类？
类是模板，类具有相同属性和行为。
属性 -> 变量，称为成员变量
行为 -> 方法，称为成员方法【行为方法参数中一定要有 self！！！！】

# 什么是对象？
根据类模板创建的具体实例 -> 对象

总结：类是对象的抽象化（模板化），对象是类的具体实例。
```


## 面向对象

为什么要创建对象？什么是面向对象？
        ***类只是一种程序内的“设计图纸”，需要基于图纸生产实体(对象)，才能正常工作，这种套路称之为：面向对象编程***

```python
# 什么是面向对象呢？
面向对象编程，就是使用对象进行编程。
即，设计类，基于类创建对象，并使用对象完成具体的工作。
```

**需求：**

- ​	设计图纸 （闹钟类）：
- ​		属性：编号、价格
- ​		行为：响铃

-    实例化：
- ​	生产三个实体（对象）

```python
# 类名首字母大写
class NaoZhong():
    # 声明两个空的成员变量 --> 属性
    id = None     # None 是空的意思
    jiaGe = None
    # 声明一个成员方法 --> 行为
    def xiangLing(self):
        # 实现真的让电脑响起来 --> 可以使用winsound 内置库
        import winsound
        # Beep(声音频率,持续时间)
        winsound.Beep(2000,3000)
    def xiangLing2(self,pingLu,shiJian):
        # 实现真的让电脑响起来 --> 可以使用winsound 内置库
        import winsound
        # Beep(声音频率,持续时间)
        winsound.Beep(pingLu,shiJian)

# 根据写的闹钟图纸创建三个闹钟
n1 = NaoZhong()     # 第一个闹钟
# 面向对象编程
n1.id = "001"     # 设置编号
n1.jiaGe = 1688     # 设置价格
print(f"n1闹钟，id是{n1.id}，价格是{n1.jiaGe}")
# n1.xiangLing()     # 让n1闹钟响铃
n1.xiangLing2(2000,3000)

n2 = NaoZhong()     # 第二个闹钟
n3 = NaoZhong()     # 第三个闹钟
```

## 构造方法

思考：
        以上闹钟类代码，从创建对象到成员变量的赋值需要依次进行，略显繁琐。有没有更加高效的方式？能够一行代码完成呢？
Clock()这个括号，能否像函数那样，通过传参的形式对属性进行赋值呢？

```python
Python类可以使用__init__()方法，称之为构造方法。
可以实现：
1、在创建对象的时候，会自动执行。
2、在创建对象的时候，将传入参数自动传递给__init__()方法使用。

注意事项：
1、构造方法名称：__init__()，千万不要忘记init前后都有2 个下划线。
2、构造方法也是成员方法，不要忘记在参数列表中提供self。
3、在构造方法内定义成员变量，需要使用self关键字。（成员变量通过self使用）
```

```python
# 类名首字母大写
class NaoZhong():
    # 声明两个空的成员变量 --> 属性 --> 写了构造方法之后可以省略

    # 创建对象的时候会优先执行init方法，执行的时候拿到创建对象的属性，然后赋值
    # 构造方法的作业，是让程序员在面向对象操作的时候，设置对象属性更加方便
    def __init__(self,id,jiaGe):
        self.id = id         # id是从外部传进来的，赋值到 self.id 中
        self.jiaGe = jiaGe
        print("__init__方法默认自动执行，不在外部使用！")

    # 声明一个成员方法 --> 行为
    def xiangLing(self):
        import winsound
        winsound.Beep(2000,3000)

    def xiangLing2(self,pingLu,shiJian):
        import winsound
        winsound.Beep(pingLu,shiJian)

# 面向对象编程
# 根据写的闹钟图纸创建三个闹钟
n1 = NaoZhong("001",1688)
print(f"n1闹钟，id是{n1.id}，价格是{n1.jiaGe}")
# n1.xiangLing()
# n1.xiangLing2(2000,3000)

n2 = NaoZhong("002",1306)     # 第二个闹钟
print(f"n2闹钟，id是{n2.id}，价格是{n2.jiaGe}")
n3 = NaoZhong("003",1573)     # 第三个闹钟
n3.xiangLing2(500,6000)
```
